Můžete si je prohlížet přímo [zde v adresářích](/mobeetle/programovani_pro_pravniky_2019/tree/master/priklady) nebo s nimi experimentovat na [JSFiddle](https://jsfiddle.net/). Klinutím na odkazy níže se příklad otevře v editoru na JSFiddle.
